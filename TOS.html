Veri. TOS and Privacy Policy

Veri. — Privacy Policy:

Last Updated: 19 June 2026

1. Information We Collect

Veri. only collects the minimum data required to provide security and verification features. This includes: User Data Discord User ID Verification status Captcha attempts (success/fail count) Honeypot triggers Timestamps of verification Server IDs where verification occurred Server Data Server ID Verification channel ID Honeypot channel ID Logs channel ID Verification role ID Admin role ID Server security settings Global Blacklist Data Discord User IDs of globally banned users Reason (honeypot trigger or manual action) Timestamp of blacklist entry

2. How We Use This Data

Veri. uses collected data only for: Verifying users Detecting suspicious or automated accounts Preventing raids and spam Enforcing server security settings Maintaining the global blacklist Providing server logs to administrators Restoring server configuration after restarts We do not use data for advertising, profiling, or selling to third parties.

3. Data Storage & Security

All data is stored locally in Veri.’s private database (veri-data.json). No data is shared with external services. No data is transmitted outside Discord except for backups made by the owner. Only the bot owner has access to the raw data.

4. Data Retention

Data is kept only as long as necessary for security purposes. Verification records remain until manually deleted or reset. Blacklist entries remain until manually removed. Server configuration remains until the server removes the bot. If a server removes Veri., all server‑specific data is automatically orphaned and can be deleted by the owner.

5. User Rights

Users may request: Removal from the global blacklist Removal of their verification data Information about what data Veri. stores about them Requests must be made by messaging the bot owner directly.

6. Children’s Privacy

Veri. does not knowingly collect data from users under 13. If such data is discovered, it will be deleted upon request.

7. Third‑Party Sharing

Veri. never shares data with: Advertisers Data brokers Analytics companies Third‑party APIs The only exception is when required by Discord’s Terms of Service or legal obligations.

8. Changes to This Policy

The owner may update this policy at any time. Continued use of Veri. means you accept the updated policy.
